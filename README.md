#Amal Counter Web App

A simple web-based application to count daily good deeds (amal). This project was built using JavaScript as part of my learning journey in web development.

##Features

- Increment and decrement count
- Simple and clean interface
- Real-time update
- Easy to use in browser

##Tech Stack

- HTML
- CSS
- JavaScript (Vanilla JS)

##Purpose

This project was created to practice:

- JavaScript basic logic
- DOM manipulation
- Event handling
- Building simple interactive applications

##How to Run

1. Download or clone this repository
2. Open
3. Use the buttons to count your daily good deeds
